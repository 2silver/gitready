Vous lisez cette page pour uen de ces raisons:

* Vous utilisez SVN depuis longtemps et vous voulez savoir comment fonctionne Git
* Vous voulez savoir si il est pénible de migrer vers Git
* Vous détestez SVN et vous voulez voir ce qui se fait de mieux.

Au lieu d'ajouter à l'ensemble des didaticiels existants sur le sujet, je me suis dit que l'astuce qui vous serait le plus utile (et <s>facile</s> pour moi) serait de lister les bons didacticiels sur la manière de migrer vers Git pour les utilisateurs de SVN.
Si vous avez plus de liens, merci de les préciser dans les commentaires, ils seront ajoutés.

*Didacticiels pour la transition*
_Ces didacticiels décrivent les différences entre les deux systèmes de gestion de version et leurs utilisations respectives à un assez haut niveau d'abstraction.

* "l'organisation du travail avec Git & SVN":http://andy.delcambre.com/2008/03/04/git-svn-workflow.html
* "Comapraison Git Svn":http://git.or.cz/gitwiki/GitSvnComparsion
* "Une introduction à git-svn pour les utilsateurs et les déserteurs de subversion/SVK":http://utsl.gen.nz/talks/git-svn/intro.html

*Didacticiels pour les commandes*
_Ces didacticiels font le point sur les commandes qui ont la même action entre les deux systèmes._

* "Git - SVN Cours intensifs":http://git.or.cz/course/svn.html
* "Comment utiliser Git et svn ensemble":http://flavio.castelli.name/howto_use_git_with_svn
* "git[-svn] en 30 minutes":http://michael-prokop.at/blog/2007/12/03/git-svn-in-30-minutes/

*Didacticiels pur les migrations*
_Ces didacticiels nous montrent comment vous pouvez importer votre historique de versions depuis SVN et comment travailler avec d'autres personnes toujours sur SVN._

* "Migrer un dépôt Subversion en un dépôt Git":http://weblog.redlinesoftware.com/2008/2/24/converting-subversion-repositories-to-git
* "Migrer un dépôt Subversion partagé vers Git":http://notahat.com/posts/25
* "Comment migrer de Subversion vers Git":http://pauldowman.com/2008/07/26/how-to-convert-from-subversion-to-git/
* "git-svn est une passerelle addictive":http://www.robbyonrails.com/articles/2008/04/10/git-svn-is-a-gateway-drug
