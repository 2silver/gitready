Você provavelmente esta lendo isto por uma das seguintes razões:

* Você já utiliza a um logo tempo SVN e quer ver se GIT é tudo isso mesmo 
* Você quer saber qual será o tamanho da dor para mudar para GIT
* Você odeia SVN e busca encontrar algo melhor

Em vez e contribuir para a massa de tutoriais já escritos sobre o assunto, pensei que seria mais <s>fácil</s> e também uma maneira mais útil compor uma lista dos grandes tutoriais que tratam sobre como realizar uma migração para o GIT para aqueles que já são mais acostumados com SVN. Se você tiver mais links, por favor comente que irei adicioná-lo à lista.

*Tutoriais de Transição*
_Estes tutoriais descrevem quais as diferenças entre os sistemas de controle de versão e sobre a forma como eles trabalham (boa parte) em alto nível._

* "Git SVN Workflow":http://andy.delcambre.com/2008/03/04/git-svn-workflow.html
* "GitSvnComparsion":http://git.or.cz/gitwiki/GitSvnComparsion
* "Uma introdução ao git-svn para usuários Subversion/SVK e desertores":http://utsl.gen.nz/talks/git-svn/intro.html

*Tutoriais de Comandos*
_Estes tutoriais focam no aprendizado de comandos que executam ações comuns entre os dois sistemas._

* "Git - SVN Crash Course":http://git.or.cz/course/svn.html
* "Como utilizar Git e SVN juntos":http://flavio.castelli.name/howto_use_git_with_svn
* "git[-svn] em 30 minutos":http://michael-prokop.at/blog/2007/12/03/git-svn-in-30-minutes/

*Tutoriais de Migração*
_Estes tutoriais ajudam a elaborar uma forma de você importar seu histórico do SVN e como trabalhar com outras pessoas que não podem realizar a conversão ainda._

* "Convertendo repositórios Subversion para Git":http://weblog.redlinesoftware.com/2008/2/24/converting-subversion-repositories-to-git
* "Convertendo um repositório Subversion compartilhado para Git":http://notahat.com/posts/25
* "Como converter de Subversion para Git":http://pauldowman.com/2008/07/26/how-to-convert-from-subversion-to-git/
* "git-svn is a gateway drug":http://www.robbyonrails.com/articles/2008/04/10/git-svn-is-a-gateway-drug

Dicas relacionadas a git-svn são ainda muito bem bem-vindas,  seja sobre configuração de opções, bugs/problemas decorrentes, etc. "Envie sempre":/submit.html que você tiver uma boa idéia!
