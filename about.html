Sobre

Se você estiver procurando por ponteiros (sem trocadilhos) sobre como usar o GIT, não procure mais! O objetivo com este site é torná-lo utilizável como um caminho para iniciantes se sairem bem no uso do Git e mostrar para aqueles que já estão familiarizados com Git alguns truques novos.

Quer mudar o layout, design ou qualquer outra coisa? Então visite a página do projeto no GitHub. Quer enviar a sua dica? Envie aqui.

Quer mandar um alô para o autor? Envie um email para nick@quaran.to, siga no Twitter ou envie uma mensagem no GitHub.

Um pequeno aviso sobre as dicas: O autor não pretende ser impecável ou perfeito de alguma forma na suas habilidades com GIT. A principal razão quando iniciei este blog era aprender mais sobre o GIT e como ele funciona. Este é um processo de aprendizagem para todos os envolvidos, e é por isso que ele é valioso tanto para os leitores quanto para os autores. Se você ver algum erros ou imprecisão, não hesite em contactar Nick ou deixar um comentário.

E para finalizar... muito obrigado pela leitura!
